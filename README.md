# WarThunder_Map_Marker_Finder

Дальномер по желтой метке на карте. Использует скриншоты и оверлей.
Основан на нейросетях

## Установка

- Скачайте архив
- Разместите папку distance на рабочем столе
- Читайте Инструкцию.txt
